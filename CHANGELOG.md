# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
e este projeto segue [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-06

### Adicionado
- Implementação completa do TechnicalDocumentTranslator
- Suporte a tradução de texto e documentos
- Custom Translator para modelos personalizados
- Glossários técnicos com suporte TSV/CSV
- Integração com Azure Blob Storage
- Avaliação de qualidade com BLEU scores
- Exemplos práticos de uso
- Documentação completa
- Testes unitários
- Scripts de automação

### Recurso
- Tradução neural baseada em transformers
- Suporte a 100+ idiomas
- Preservação de formatação de documentos
- Conformidade LGPD/GDPR
- Managed Identity para segurança
